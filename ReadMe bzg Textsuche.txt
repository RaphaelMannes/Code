Entstanden als Tool zur Überwachung von .log-Dateien von Minecraft-Servern.
Aufgegeben zusammen mit dem Server aus finanziellen Gründen.

- Findet Warungen mit dem Button "Warnings"
- Findet Serverinfos, wie zB. Speichern der Weltdaten/starten des Servers oder sogar wichtige Regeländerungen
	des Servers mit dem button "infos".
- Findet JEDEN String mit dem "Search-String" button. Groß-klein schreibung wird misachtet.


- Export button sowie Zeit-Suche funktioniert nicht
