STT (Speech to text) in Java

- Grundstock von tutorial und Api, Programm konnte nur zahlen verstehen
- eigene Anpassung auf befehle (grammars/grammar.gram  (notepad) )
- theoretisch sind alle englischen Wörter möglich --> führt zu fehlern (bank, tank - tree, three, we)


Besipiele
============================

google dog		--> öffnet google und sucht nach Dog ~ Hund
open / open notepad	-->  öffnet programme / notepad	  (NUR notepad geht bisher)
type (dog)		-->  schreibt
